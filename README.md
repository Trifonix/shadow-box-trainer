# Веб-приложение тренажёр-таймер для самостоятельных тренировок по боксу

Приложение делает:

1) 10 минут таймер с обратным отсчётом упражнение "скакалка" или "прыжки на месте если нет скакалки"

2) 2 минуты отдых - таймер с обратным отсчётом

3) 10 минут упражнение "бой с тенью"

4) 2 минуты отдых - таймер с обратным отсчётом

5) 10 минут упражнение "отжимания"

6) 2 минуты отдых - таймер с обратным отсчётом

7) 10 минут упражнение "приседания"

8) конец тренировки

Приложение на JavaScript с красивым современным UI для деплоя на Vercel.

Содержит 1 кнопку "СТАРТ" в начале тренировки и кнопку "ПАУЗА" в процессе, если понадобится прерваться.

Полностью состоит из красивых анимаций нативных объектов JavaScript - демонстрация упражнений, чтобы занимающийся понимал что сейчас делать. И звуковое сопровождение, чтобы слышать когда начинается отдых и готовиться к следующему упражнению.
